# Calculadora de números complejos
Se realizo una calculadora para números complejos con las operaciones algebraicas mas conocidas y unas trasnformaciones.\
Las operaciones son:
1. **Suma**
2. **Resta**
3. **Multiplicación**
4.  **División**
5. **Conjugado**
6. **Opuesto**
7. **Modulo**
8. **Cartesianas a polares**
9. **Retornar fase**
